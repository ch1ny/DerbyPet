### 方法二、自行编译

> 仅推荐有相关知识储备的同学
>
> 默认看到这里的你已经安装了 nodejs(npm) 或 yarn

进入项目根目录，打开终端，依次执行一下命令：

#### 一、下载依赖

```powershell
# 如果您使用的是 npm
npm install
# 或者您使用的是 yarn
yarn install
```

#### 二、编译代码

```powershell
# 如果您使用的是 npm
npm run dist
# 或者您使用的是 yarn
yarn dist
```

#### 三、运行程序

编译结束后，进入项目根目录下的 `dist/DerbyPet-win32-x64` 文件夹中，双击 `DerbyPet.exe` 即可运行程序。
