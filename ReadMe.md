# Umamusume Driver

## Introduction

一款为了能将 **ウマ娘　プリティーダービー** 加入 **Steam** 游玩而使用 **electron** 与 **React.js** 制作的一款赛马娘启动器。

> 本来只是我个人做着玩玩的，没想到视频火了……
>
> 而且还有很多站友希望我能分享给大家……

目前我还没有找到如何正确做到一键启动，DMMPlayer 的源码做了混淆我读不懂 🤯。
